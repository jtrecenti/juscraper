Using filters

You can filter by date range, judge (relator), court chamber (orgao_julgador), and case type (tipo_processo).

dados_filtrados = tjmt.cjsg(
    'direito',
    paginas=1,
    data_julgamento_inicio='2025-01-01',
    data_julgamento_fim='2025-06-30',
    tipo_processo='Criminal',
)
print(dados_filtrados.shape)

Download and parse separately

You can also download the raw data first and parse it later.

brutos = tjmt.cjsg_download('direito', paginas=1)
print(f"Pages downloaded: {len(brutos)}")
print(f"Keys in response: {list(brutos[0].keys())}")

dados = tjmt.cjsg_parse(brutos)
import pandas as pd
df = pd.DataFrame(dados)
print(f"\nParsed shape: {df.shape}")
df.head(3)
